GUIDELINES TO USE

This is a demo banking web app based on frontend technologies. No attempt is made to save progress to any sort of database so progress is lost if browser is refreshed. All activities performed on the platform are only available while in use at the moment.

FEATURES AND INFO

  • Click on the icon to see all available users profile on the platform to login to or simply create an accout. However, account is lost after refreshing or reopening site.
  • There are locally stored accounts for users to interact with, they can be viewed by clicking the icon pointed to above. The icon is only availabe while logged out of accounts and currently in the login form page.
  • Transfer to all accounts present on platform including user's created account. The accounts available can be viewed by clicking on the View accounts prompt on the transfer form which can be accessed by navigating to the transfer page using the side navigations.
  • Loan can be credited to user's account, however the loan policy is 50% of user's current balance.
  • Can log out of current account and cross login between existing one or simply create new accounts. All activities can be performed on all accounts and can be checked by logging into them.
  • Accounts can also be deleted from platform. If the account deleted is the user created accounts, it cannot be recovered. However, if the account deleted is the app built account, they can simply be recovered by refreshing the browser. But all progress made to the platform is los.t
  • Funny sapa alert is added to the platform if balance is lower than specific amount.
  • To Locate this prompt again, it can be accessed by clicking the help section under the settings page
  • Settings > Help
  • Platform will constantly be updated stay tuned.
  • This is not a real banking app!
